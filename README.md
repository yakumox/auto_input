# Auto Input – 開発者向けWebフォーム自動入力ツール

フォームへの自動入力を行う Chrome 拡張機能です。  
複数の**プロファイル**に入力パターンを保存し、ボタン1つでフォームを一括入力できます。

---

## 機能

| 機能 | 説明 |
|------|------|
| **入力** | 選択中のプロファイルに保存された値をページのフォームへ一括入力する |
| **記憶** | 現在のページの入力内容・選択内容をプロファイルに保存する |
| **設定** | プロファイルの作成・削除・フィールド設定を行う設定画面を開く |
| **プロファイル** | 複数の入力パターンをプロファイルとして管理する |

### 対応フィールドタイプ

- `input[type="text"]` / `email` / `password` / `tel` / `number` / `date` など
- `textarea`
- `select`（option の value または表示テキストで選択）
- `input[type="radio"]`（グループ内の特定のラジオボタンを選択）
- `input[type="checkbox"]`（チェックのオン/オフ）

> `input[type="file"]` はブラウザのセキュリティ制限により非対応です。

## インストール

1. このリポジトリをクローン、または ZIP ダウンロードして解凍する

   ```bash
   git clone https://github.com/yakumox/auto_input.git
   ```

2. Chrome を開き、アドレスバーに `chrome://extensions/` と入力する

3. 右上の **「デベロッパーモード」** を有効にする

4. **「パッケージ化されていない拡張機能を読み込む」** をクリックし、リポジトリのフォルダを選択する

## 使い方

### 1. プロファイルを作成する

拡張機能アイコンをクリックしてポップアップを開き、プロファイルのセレクトボックスから **「＋ 新規作成...」** を選択してプロファイル名を入力します。  
または **「設定」** ボタン → 設定画面の **「＋ 新規作成」** から作成できます。

### 2. 入力パターンを設定する（手動）

**「設定」** ボタンをクリックして設定画面を開き、プロファイルを選択してフィールドを追加します。

| 列 | 説明 |
|----|------|
| ラベル | フィールドの識別名（任意） |
| タイプ | フィールドの種類（text / select / radio / checkbox など） |
| CSS セレクタ | 入力対象要素を特定する CSS セレクタ（例: `#email`、`input[name="username"]`） |
| 値 | 入力する値（radio の場合は `value` 属性の値、checkbox の場合は `true` / `false`） |

設定後、**「💾 保存」** をクリックして保存します。

### 3. 入力パターンを記憶する（自動）

入力したいフォームページを開き、値を手動で入力・選択した状態にしてから、ポップアップの **「💾 記憶」** ボタンをクリックします。  
現在のページのすべての入力内容がプロファイルに保存されます（既存の設定は上書きされます）。

### 4. 自動入力する

ポップアップでプロファイルを選択し、**「▶ 入力」** ボタンをクリックします。  
ページの全フィールドがクリアされた後、プロファイルのパターンにしたがって一括入力されます。

## ファイル構成

```
auto_input/
├── manifest.json          # 拡張機能マニフェスト (Manifest V3)
├── icons/
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
├── popup/
│   ├── popup.html         # ポップアップ UI
│   ├── popup.css
│   └── popup.js           # ポップアップのロジック
├── options/
│   ├── options.html       # 設定画面 UI
│   ├── options.css
│   └── options.js         # 設定画面のロジック
└── content/
    └── content.js         # コンテントスクリプト（入力・収集処理）
```

## 技術仕様

- **Manifest V3** ベースの Chrome 拡張機能
- **Storage**: `chrome.storage.local` でプロファイルを永続保存
- **通信**: `chrome.tabs.sendMessage` でポップアップ ↔ コンテントスクリプト間通信
- **React 等フレームワーク対応**: `HTMLInputElement.prototype` のネイティブ setter を使用して `input` / `change` イベントを発火
- **radio ボタン**: `input[type="radio"][name="X"]` セレクタでグループを対象とし、`value` 属性が一致するものをチェック
- **checkbox**: `display:none` で隠された要素にも対応（`offsetParent` チェックを除外）
